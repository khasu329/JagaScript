## 命令セット
この言語では、標準で(メモリ領域の限り)無限の配列が用意されています。
[]に挟まれた数値は、配列の添え字として機能します。また、ここでのアドレスは、命令の何文字目かを示しています。
| 命令 | 形式 | 処理 | 備考 |
| :---: | :---: | :--- | :--- |
| a | `a j H1 j H2` | `[H1] = [H1] + [H2]` | 加算。H2 の値を使用。 |
| m | `m j H1 j H2` | `[H1] = [H1] - [H2]` | 減算。H2 の値を使用。 |
| i | `i j H j S` | `[H] = S` | 即値 S を代入。配列 間のコピーには使えません。 |
| om | `om j H` | `[H]` の値を数値として出力。 | |
| o | `o j H` | `[H]` の値を ASCII 文字として出力。 | |
| g | `g j H1 j (比較演算子の区別用文字) j H2` | `[H1]と0`を比較し、条件が真のときアドレス H2 にジャンプする。 | 区別用文字がaのとき、等値演算子として、iのとき非同値演算子として、mのとき大なり演算子として、oのとき小なり演算子として、振舞います。 |
##
これで書いたコードをスペースや改行なしで連結することで動かすことができます。
